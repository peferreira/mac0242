O nosso jogo foi feito utilizando a plataforma eclipse, portanto o projeto pode ser
facilmente importado para o eclipse. A Classe main está no arquivo Gerenciador.java
e o arquivo do java cc está no diretorio parser, na classe Parser.jj.

Tivemos alguns problemas com tempo, e infelizmente para essa fase, nosso parser ficou incompleto.

WHILE: funciona mas pode dar erros para alguns casos.
FUNÇÕES: não foram implementadas.

de resto, é possível criar alguns códigos.

Os programas são constrúidos através de códigos que estão na pasta parser no diretorio do
jogo. (por exemplo: programa1.txt será para o jogador 1, programa2.txt será para o jogador 2).

Pode-se notar que na fase 3 conseguiamos implementar
